# Desynced AMP Template

AMP template for the Desynced Dedicated Server.

## Steam AppID
Server: 2943070

## Default Ports
- Game Port: 7777 UDP
- Query Port: 27015 UDP

## Install
Copy template files into:

AMPData/Plugins/ADSModule/GenericTemplates/

Restart AMP and create a new instance using the Desynced template.
